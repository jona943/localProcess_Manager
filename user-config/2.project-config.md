# Requisitos y Estándares del Proyecto (Project Config)

Define aquí el stack tecnológico, arquitectura, estándares del proyecto y sus archivos de contexto.

| Propiedad | Valor | Descripción |
| :--- | :--- | :--- |
| **Nombre del Proyecto** | [Mi-Proyecto] | El nombre del proyecto de desarrollo. |
| **Directorio del Proyecto** | [../ruta-del-proyecto] | Ruta relativa o absoluta al repositorio de código fuente. |
| **Archivo de Contexto** | [proyecto-contexto.md] | Nombre del archivo markdown que contiene la base de conocimiento técnica. |
| **Tecnologías Principales** | Node.js, JavaScript | El stack tecnológico base del proyecto. |
| **Módulos de JavaScript** | ES Modules (import / export) | Sistema de carga de módulos (CommonJS o ES Modules). |
| **Estilos (CSS)** | Vanilla CSS | Framework o metodología CSS (Vanilla, Tailwind, Sass). |
| **Manejo de Asincronía** | Asíncrono puro (async/await) | Estilo preferido para I/O y llamadas de red. |
| **Arquitectura de Código** | Modular | Estilo arquitectónico sugerido para organizar el software. |
| **Comentarios en Código** | Español sencillo | Idioma de los comentarios agregados al código fuente del proyecto. |
